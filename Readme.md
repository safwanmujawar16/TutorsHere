## Technology Stack 
Frontend: ASP Controls, HTML, CSS, JavaScript, Bootstrap
Backend: ASP.NET Framework (C#)
Programming Language: C#
Database: SQL Server Management Studio
Developing Platform: Microsoft Visual Studio
